# Tank 2025

## Description
This project is a 2D tank game developed using JavaFX. The game includes a player-controlled tank and enemy tanks, featuring movement, shooting mechanics, and real-time interactions.

The system provides a graphical user interface and simulates gameplay elements such as collisions, scoring, and game state management.

## Features
- Player movement in four directions
- Shooting mechanics with bullet physics
- Enemy AI with random movement and shooting
- Collision detection with walls and tanks
- Score and life tracking system
- Game over and restart functionality
- Pause menu with control options

## Concepts Used
- JavaFX (GUI development)
- Event handling and user input
- Game loop and real-time updates
- Object-Oriented Programming

## Technologies
- Java
- JavaFX

## How to Run
```bash
javac Tank2025.java
java Tank2025
```
