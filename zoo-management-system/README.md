# Zoo Management System

## Description
This project is a simulation of a zoo management system developed using object-oriented programming principles. It manages animals, personnel, and visitors, and processes operations such as feeding animals and visiting habitats.

The system reads input from files and handles different scenarios using custom exception handling.

## Features
- Manage different types of animals with specific behaviors
- Implement feeding and habitat cleaning operations
- Handle user roles (personnel and visitor) with different permissions
- Use custom exceptions for error handling (e.g., insufficient food, invalid user actions)

## Concepts Used
- Object-Oriented Programming (Abstraction, Polymorphism)
- Exception Handling
- File I/O

## Technologies
- Java
- OOP Design Principles

## How to Run
```bash
javac Main.java
java Main animals.txt persons.txt foods.txt commands.txt output.txt
```
